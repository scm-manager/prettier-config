<h1 align="center">
  @scm-manager/prettier-config
</h1>

[Prettier](https://prettier.io/) configuration for SCM-Manager related projects.

## Installation

Install the `@scm-manager/prettier-config` as dev dependency:

```bash
yarn add --dev @scm-manager/prettier-config
# or 
npm install --save-dev @scm-manager/prettier-config
```

Configure the preset by adding the following snippet to your `package.json`:

```json
"prettier": "@scm-manager/prettier-config"
```
